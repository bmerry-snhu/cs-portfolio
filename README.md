# Computer Science & Information Technology Portfolio

Welcome! This repository contains selected coursework, projects, and technical artifacts completed throughout my Computer Science degree program at Southern New Hampshire University (SNHU), as well as previous Information Technology and Cybersecurity coursework.

My goal is to use this repository as a living portfolio that documents my growth as a software developer, systems engineer, and cybersecurity professional. Wherever possible, projects are preserved in a clean, buildable state with their original documentation and supporting materials.

## Areas of Study

This repository includes work from courses covering topics such as:

- Software Development
- Object-Oriented Programming
- Operating Systems
- Computer Networking
- Information Security
- Cybersecurity
- Systems Administration
- Databases
- Software Engineering
- Data Structures and Algorithms

## Repository Organization

Each course has its own directory containing the relevant source code, documentation, diagrams, and project deliverables.
